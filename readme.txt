## Optimize your Google Ads Campaigns - Improve your ROI with Google Shopping, Display, Re-marketing & Serach Ads##
This Google Adwords plugin is available for any WordPress or WooCommerce site that is spending money on Google Ads. 
It simply connects to your Google Ads account and allows you have to have a dashboard within your WordPress site with all your campaign details. Not only does it make it easy for you to keep an eye on your Google Ads campaigns, it will analyze the campaign data and start to make recommendations that you can either apply or not, to run better campaigns with better results.

== Installation ==
This section describes how to install the plugin and get it working.
1. Download and install the plugin from WordPress dashboard. You can also upload the entire “Muzaara-Lite-WP” folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to Muzaara on left-hand side menu in WordPress and click Settings:
4. Sign In With Google to link your Google Ads data with the plugin.
5. Chose if you want the dashboard to show in main WP home page or not and click save.
6. The dashboard will now show on your mian WP dashboard, reccomednations will start to show when our system has process enough data to make suggestions.



== Frequently Asked Questions ==
= Will the plugin make any changes to my campaigns? =
No, it will show your recommendations on what to change to run better campaigns, you can decide to action these directly from the plugin, but it will not make any changes without your permission.

= What sort of Adwords campaigns will this work with? =
It will work with any campaigns that you create in Google Ads.

= How will this plugin optimize my Google Ads? =
Each recommendation provides customised suggestions to help increase your campaigns' performance. Recommendations can introduce you to new, relevant features, help you get more out of your budget by improving your bids, keywords and ads, and can work to increase the overall performance and efficiency of your campaigns.
